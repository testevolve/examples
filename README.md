# Example projects
These are some projects to help you build out your tests using TestEvolve
## How to use
1. fork this repository
2. follow steps for either the ruby or javascript framework

### JavaScript
In the Javascript folder there are two parts: The **demos** folder (which holds example features with their steps and pages) and the **template** folder (which holds the testevolve code).

1. Move which ever demo project /features you like from the demos folder into template/features (this will merge)
eg. documentation/features -> template/features

2. `npm install` from inside the template folder

3. `npm run test` to run the test features

### Ruby
In the Ruby folder there are two parts: The **demos** folder (which holds example features with their steps and pages) and the **template** folder (which holds the testevolve code).

1. Move which ever demo project /features you like from the demos folder into template/features (this will merge)
eg. browser-local-site/features -> template/features

2. `bundle install` from inside the template folder

3. `cucumber` to run test features
